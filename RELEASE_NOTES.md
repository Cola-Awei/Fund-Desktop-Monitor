# Release Notes

## v0.1.0

首个公开版本。

### 功能

- Windows 桌面 exe 小窗口
- 深色科技风界面
- 基金代码、持有份额、持仓成本单价添加持仓
- 每 30 秒自动刷新实时估值
- 当日实时盈亏汇总
- 红涨绿跌显示
- 窗口支持拖动、缩放、最小化
- 点击单个基金查看关联股票持仓明细
- 股票详情展示实时涨跌、持仓占比、较上期占比
- 本地持久化持仓数据

### 数据说明

- 基金估值数据来自天天基金公开接口
- 基金持仓和股票行情来自东方财富/天天基金公开接口
- 盘中估值仅供参考，最终数据以基金公司和交易平台为准

### 验证

- `npm run package:installer`
- 71 个自动化测试通过

### Windows 安装版

推荐下载 GitHub Release 附件中的安装程序：

```text
Fund Desktop Monitor Setup 0.1.0.exe
```
