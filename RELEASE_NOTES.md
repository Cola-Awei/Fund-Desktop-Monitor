# Release Notes

## v0.1.1

界面体验修复版本。

### 修复与优化

- 点击右上角刷新按钮后，立即显示“正在刷新...”，刷新图标旋转并临时禁用按钮，避免重复点击和无反馈感。
- 手动刷新完成后，底部显示“已刷新 HH:mm”，并立即应用最新快照。
- 底部时间改为当前实时时间，不再用估值时间冒充当前时间。
- 估值时间达到 15:00 后，汇总区显示“已闭市 15:00”。
- 基金关联股票弹窗顶部只显示“截止日期”，不再显示“对比日期”。

### 验证

- `npm test`
- `npm run package`
- `npm run package:installer`
- 73 个自动化测试通过

### Windows 安装版

推荐下载 GitHub Release 附件中的安装程序：

```text
Fund Desktop Monitor Setup 0.1.1.exe
```

## v0.1.0

首个公开版本。

### 功能

- Windows 桌面 exe 小窗口
- 深色科技风界面
- 基金代码、持有份额、持仓成本单价添加持仓
- 每 30 秒自动刷新实时估值
- 当日实时盈亏汇总
- 红涨绿跌显示
- 窗口支持拖动、缩放、最小化
- 点击单个基金查看关联股票持仓明细
- 股票详情展示实时涨跌、持仓占比、较上期占比
- 本地持久化持仓数据

### 数据说明

- 基金估值数据来自天天基金公开接口
- 基金持仓和股票行情来自东方财富/天天基金公开接口
- 盘中估值仅供参考，最终数据以基金公司和交易平台为准

- 71 个自动化测试通过
